LFF Linux Package Repository

Official package archive for LFF Linux

Add this repository

Run this one-liner on a Debian or Ubuntu system:

curl -fsSL https://lff-linux.github.io/addrepo.sh | sudo bash

The script checks for apt-get, installs the repository key, and adds the signed APT source entry.
